Objetivo

Prever e avaliar a miscibilidade entre vários líquidos


Método

Previsão

Para fazermos uma previsão sobre se dois líquidos são miscíveis entre si ou não, iremos usar uma regra geral, que diz que:


Dois líquidos são miscíveis se:

  • Ambos forem compostos por moléculas polares.
  • Ambos forem compostos por moléculas apolares.

Caso nenhuma das condições se verifique, os líquidos são imiscíveis.


Testar a Miscibilidade

Usando um tubo de ensaio, visto ser estreito, transparente e facilitar a agitação, apenas iremos usar alguns mililitros de cada líquido, para verificar se estes são miscíveis.

Após escolhermos dois líquidos, iremos usar um conta-gotas diferente para cada um. Após adicionarmos algumas gotas de cada um dos líquidos e agitarmos vigorosamente o tubo de ensaio, ir-se-à verificar uma de duas situações:

  • Formaram-se duas camadas → Os líquidos são imiscíveis
  • Apenas há uma camada → Os líquidos são miscíveis

Se ambos os líquidos forem incolores, adicionamos corante a um deles, de forma a percebermos melhor se, depois da agitação, estão presentes duas camadas ou apenas uma.
